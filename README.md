# Project Manager Simulator

## Overview
The Project Manager Simulator is a web-based game where you take on the role of a software project manager, overseeing a team of diverse professionals to complete tasks in order to see a project to completion. The game allows users to manage tasks, track project progress, and make decisions that affect project outcomes. It is designed to demonstrate concepts related to project planning, decision-making, and software development workflows.

## Features
- User authentication (sign up, log in, log out)
- Create a variety of teams
- Interactive project dashboard
- Managing Team Members to work or rest depending on their happiness, energy, and work-life balance.
- Task and decision management
- Progress tracking
- Bug/progress mechanics that respond to user decisions
